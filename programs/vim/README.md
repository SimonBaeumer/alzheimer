# vim

syntax verb + noun

## Misc

 - `set paste` copy into vim without formatting
 - `v` visually select
 - `>` indent
 - `c` change (delete and enter insert mode)
 - `crtl+r` redo (undo undo)
 - `j` lines down

 ## Text objects

 - `iw` inner word
 - `it` inner tag
 - `iq` inner quotes
 - `ip` inner paragraph
 - `as` a sentence
